Preguntas frecuentes

Preguntas que pueden ayudarte antes de cotizar.

¿Puedo llegar sin tener todo claro?
Sí. Podés llegar con una idea inicial y la ordenamos según el servicio que mejor se adapte a tu necesidad.
¿Cómo sé cuál servicio necesito?
Podés escribirme contando qué querés lograr. A partir de eso te recomiendo el camino más adecuado.
¿Cómo funciona el manual de marca?
Construimos una identidad visual completa: logo, variaciones, paleta, tipografías, archivos finales y aplicaciones listas para usar.
¿Las invitaciones digitales son personalizadas?
Sí. Se crean tipo página web y pueden incluir información, ubicación, confirmación, itinerario, menú, cuenta regresiva y más.
¿Cómo se cotizan las páginas web?
Según objetivo, secciones, funciones, textos, diseño y nivel de complejidad del proyecto.
¿La reunión inicial se descuenta?
Sí. En documentos, webs o proyectos que requieren análisis previo, los ₡20.000 se descuentan cuando el proyecto se realiza.
¿Puedo pedir algo diferente a la lista?
Sí. Podemos revisar la necesidad y ver si se adapta a una de las líneas de trabajo o cotizarlo aparte.

¿Lista para empezar?

Si tenés una idea dando vueltas, podemos darle forma.

A veces el primer paso es ordenar. A veces es diseñar, fotografiar, escribir o construir una página. Contame qué necesitás resolver.
